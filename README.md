# FinalProject
Wikipedia Web Traffic (Kaggle) Ежедневное количество просмотров статей Википедии за 2018-2019 годы.
Проект посвящён комплексному анализу временных рядов посещений страниц Wikipedia с применением методов машинного обучения для прогнозирования трафика и обнаружения аномалий.
Датасет
Источник: Wikipedia Web Traffic (Kaggle)
Формат: CSV файл с колонкой Page и ежедневными значениями трафика
Структура: строки — страницы Wikipedia, столбцы — даты
Период: 2015-2016 годы
Особенности: наличие пропущенных значений, различная популярность страниц
